¿Cómo obtengo mi xPub en Ubuntu??

Para obtener su xPub en KeepKey con Ubuntu, complete los siguientes pasos:
I. Instale pip

- Abra Terminal.
- Ingrese el siguiente comando: sudo easy_install pip

II. Instale Cython

- Abra Terminal.
- Ingrese el siguiente comando: sudo pip install cython

III. Instale Libsub

- Abra Terminal.
- Ingrese el siguiente comando: sudo pip install libusb-1.0.0-dev

IV. Instale KeepKey Python Client

- Abra Terminal.
- Ingrese el siguiente comando: sudo pip install HTTPS://GITHUB.COM/KEEPKEY/PYTHON-KEEPKEY/ARCHIVE/MASTER.ZIP


V. Recupere xpub key

- Abra Terminal.
- Ingrese el siguiente comando: keepkeyctl get_public_node -n "44 '/ 0' / 0 '"
- Ingrese el PIN de acuerdo con la pantalla KeepKey y la ubicación de los números en el símbolo del sistema
