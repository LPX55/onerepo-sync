Estoy en Windows 7 y tengo problemas con mi KeepKey. ¿Puede usted ayudar?

Si tiene problemas con su KeepKey y está utilizando Windows 7, le recomendamos que actualice su sistema operativo a Windows 10. Windows esta eliminando el soporte para Windows 7,y debido a esto, recomendamos usar Windows 10 para KeepKey y ShapeShift.com. Si intenta utilizar la plataforma ShapeShift o KeepKey Chrome Client con Windows 7, nuestro equipo puede verse limitado en nuestras capacidades de solución de problemas.

Si necesita ayuda para actualizar a Windows 10, esta lista de preguntas frecuentes puede ser de ayuda.
