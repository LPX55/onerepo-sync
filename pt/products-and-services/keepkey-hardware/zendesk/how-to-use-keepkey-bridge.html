Como utilizar o aplicativo KeepKey Bridge

O que é KeepKey Bridge?

O KeepKey Bridge é um aplicativo do Microsoft Windows desenhado para solucionar com problemas de conectividade webUSB entre o dispositivo KeepKey e uma máquina Windows.

Esta “ponte” é executada como aplicativo e abre uma conexão USB local para a KeepKey. A “ponte" libera uma porta localhost que o aplicativo digital ShapeShift pode utilizar para se comunicar com a KeepKey.

Para quem este aplicativo é indicado?

Se você estiver usando uma máquina Windows e estiver enfrentando para conectar sua KeepKey em beta.shapeshift.com.

Instalando o aplicativo

1. Antes de executar o aplicativo, certifique-se de que sua KeepKey esteja atualizada com o bootloader / firmware mais recente. Baixe aqui.  Agora, baixe o aplicativo KeepKey Bridge. 

Importante: Seu computador pode acusar avisos ao baixar / executar este aplicativo. Clique em "Executar mesmo assim" para continuar a instalação.

2. Depois que o arquivo kkbsetup.exe for baixado, você pode verificar se o arquivo é o que afirmamos ser, comparando o SHA256 do executável com o resumo publicado nas notas de versão. Isso pode ser feito abrindo uma janela cmd (Prompt de Comando), navegando até a pasta Downloads e digitando: cetuil -hashfile kkbsetup.exe SHA256. A informação resultante exibida deve corresponder ao que é publicado nas notas de versão do repositório python-keepkey. Se tudo estiver correto, clique duas vezes no arquivo para instalar (a criação de um atalho na área de trabalho é recomendada para sua conveniência).

Na prática

3. Ao iniciar o aplicativo KeepKey Bridge, uma janela em branco aparecerá. Mesmo que esteja em branco, ele está funcionando (sabemos que não é uma interface ideal, mas estamos trabalhando para melhorar isso). Quando a janela é fechada, o aplicativo para de funcionar.

4. Com o aplicativo aberto funcionando, vá para beta.shapeshift.com. Selecione KeepKey como a carteira que deseja emparelhar.

5. Leia os termos de serviço e política de privadade, então clique em "TRY KEEPKEY BRIDGE" para conectar.

Se for bem-sucedido, sua carteira começará a sincronizar e, eventualmente, se conectará à plataforma.

Solução de problemas

Se você visualizar alguma mensagem de erro (como a imagem acima) ao tentar emparelhar sua KeepKey por meio do aplicativo ou simplesmente não se conectar, tente o seguinte:

1. Vá até o painel de configuração e clique em “Forget". Emparelhe novamente sua KeeKey.

2. Desconecte sua KeepKey

3. Atualize a página beta.shapeshift.com.

4. Feche o aplicativo KeepKey Bridge

5. Repita as etapas da seção "Na prática" deste artigo.

 

Caso você ainda possua problemas para utilizar este aplicativo, entre em contato com a Equipe de Suporte.
