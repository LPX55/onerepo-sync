Comment utiliser KeepKey avec MyCelium ?

Une fois que vous avez initialisé votre KeepKey, vous pouvez l'utiliser en conjonction avec l'application MyCelium sur Android ou sur iOS. Suivez les étapes ci-dessous.

  1. Connectez votre KeepKey via un câble OTG (vendu séparément) à votre téléphone et ouvrez l'application MyCelium, qui est disponible sur le Google Play store et l'App Store.
  2. Sélectionnez l'icône représentant trois points dans le coin supérieur droit et appuyez sur le stockage à froid.
  3. Vous verrez que KeepKey est listé comme une option.
  4. Lorsque vous y êtes invité, entrez votre code PIN comme vous le feriez normalement.
  5. Vous pouvez maintenant utiliser votre KeepKey avec MyCelium.
