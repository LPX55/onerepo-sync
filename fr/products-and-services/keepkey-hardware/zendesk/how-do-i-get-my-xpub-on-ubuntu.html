Comment obtenir mon xPub sur Ubuntu ?

Pour obtenir votre xPub sur KeepKey avec Ubuntu, effectuez les étapes suivantes :

I. Installer pip

- Ouvrir Terminal.
- Entrer la commande suivante - sudo easy_install pip

II. Installer Cython

- Ouvrir Terminal.
- Entrer la commande suivante - sudo pip install cython

III. Installer Libsub

- Ouvrir Terminal.
- Entrer la commande suivante - sudo pip install libusb-1.0.0-dev

IV. Installer KeepKey Python Client

- Ouvrir Terminal.
- Entrer la commande suivante - sudo pip install HTTPS://GITHUB.COM/KEEPKEY/PYTHON-KEEPKEY/ARCHIVE/MASTER.ZIP


V. Récupérer la clé xpub

- Open Terminal.
- Entrer la commande suivante - keepkeyctl get_public_node -n "44'/0'/0'"
- Enter le NIP selon l'écran KeepKey et le placement des chiffres dans l'invite de commande
