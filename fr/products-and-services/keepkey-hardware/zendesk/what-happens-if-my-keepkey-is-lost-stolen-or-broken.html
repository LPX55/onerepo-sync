Que se passe-t-il si mon KeepKey est perdu, volé ou cassé ?

En cas de perte, de vol ou de bris de votre appareil, vous pouvez utiliser votre phrase de récupération de 12 mots pour restaurer tous vos actifs. Tant que vous avez un code PIN fort sur votre KeepKey, votre appareil est inutile dans les mains d'un voleur.

Si vous n'avez pas plusieurs comptes, vous pouvez récupérer vos actifs numériques sur n'importe quel porte-monnaie compatible BIP39. Si vous avez plusieurs comptes, vous pouvez les récupérer dans n'importe quel porte-monnaie compatible BIP44.

Voici quelques porte-monnaie compatibles BIP44 qui pourraient vous être utiles :

  • Exodus
  • JAXX
  • CoPay
  • Coinomi
  • MyCelium

Vous pouvez également restaurer votre phrase de récupération de 12 mots sur un autre appareil KeepKey - Veuillez suivre les instructions indiquées ici.
