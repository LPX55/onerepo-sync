Comment utiliser le KeepKey Bridge (Pont) ?

Qu'est-ce que le KeepKey Bridge ?

Le KeepKey Bridge est une application Microsoft Windows qui est conçue pour aider à résoudre les problèmes de connectivité webUSB entre KeepKey et une machine Windows.

Le pont fonctionne comme une application qui ouvre une connexion USB locale au KeepKey. Le pont expose un port localhost que l'application web ShapeShift peut utiliser pour communiquer avec la clé KeepKey. La clé KeepKey elle-même n'est pas au courant de l'existence de la passerelle, mais l'application Web ShapeShift doit être autorisée pour utiliser la passerelle.

Quand dois-je l'essayer ?

Si vous utilisez une machine Windows et que vous avez du mal à connecter votre KeepKey à beta.shapeshift.com, essayez d'utiliser la passerelle en suivant les instructions ci-dessous.

Installation du KeepKey Bridge

1. Avant d'utiliser la passerelle, assurez-vous que votre KeepKey dispose de la dernière version du bootloader/firmware. L'application de mise à jour peut être trouvée ici. Téléchargez la dernière version de KeepKey Bridge ici.

Remarque: Votre ordinateur émettra probablement quelques avertissements lors du téléchargement et de l'exécution de cette application. Vous pouvez cliquer sur “Run Anyway” (Exécuter quand même) pour poursuivre l'installation.

2. Une fois le fichier kkbsetup.exe téléchargé, vous pouvez vérifier que le fichier est bien celui que nous prétendons être en comparant le SHA256 de l'exécutable avec le code publié dans les notes de mise à jour. Pour ce faire, ouvrez une fenêtre cmd (Command Prompt), accédez au dossier “Téléchargements” et tapez : cetuil -hashfile kkbsetup.exe SHA256. Les informations affichées devraient correspondre à ce qui est publié dans les notes de version de python-keepkey. Si tout correspond, double-cliquez sur le fichier pour l'installer (il est recommandé de créer un raccourci sur le bureau pour plus de commodité).

Utilisation du KeepKey Bridge

3. Au démarrage de l'application KeepKey Bridge, une fenêtre vide apparaît. Même si elle est vide, le pont est en cours d'exécution (nous savons que ce n'est pas une interface utilisateur idéale, mais nous travaillons sur une amélioration prochainement). Lorsque la fenêtre est fermée, le pont s'arrête de fonctionner.

4. Avec le pont en marche, allez sur beta.shapeshift.com. Sélectionnez KeepKey comme porte-monnaie que vous souhaitez appairer.

5. Cliquez sur la case à cocher, puis sélectionnez "TRY KEEPKEY BRIDGE" (ESSAYER LE PONT KEEPKEY).

En cas de succès, votre porte-monnaie commencera à se synchroniser et finira par se connecter à la plateforme.

Dépannage

Si vous rencontrez des messages d'erreur (comme celui illustré ci-dessus) lorsque vous essayez d'appairer votre KeepKey via le pont ou s'il ne se connecte pas, essayez ce qui suit :

 

1. Allez à la page des paramètres et cliquez sur “Forget” (Oublier) pour oublier votre KeepKey.

2. Débranchez votre KeepKey.

3. Rafraîchissez beta.shapeshift.com.

4. Fermez l'application Bridge.

5. Répétez les étapes de la section "Utilisation du KeepKey Bridge" de cet article.

Si vous avez des questions, veuillez contacter le service clientèle ici.
