Intégration Electrum avec KeepKey

Veuillez suivre ces instructions pour intégrer Electrum à votre appareil KeepKey.

Téléchargez d'abord Electrum via ce lien : https://electrum.org/#download

Sélectionnez d'abord "Standard Wallet" (Porte-monnaie standard) :

Sélectionnez "Use a hardware device" (Utiliser un périphérique matériel) :

Connectez votre appareil KeepKey - cliquez sur "Next" (Suivant)

Sélectionnez votre appareil :

Entrez votre code PIN actuel :

Sélectionnez “Legacy (P2PKH)" :

Si vous souhaitez sélectionner un autre compte sur votre appareil KeepKey, veuillez suivre le chemin de dérivation ci-dessous :

Compte #0: m/44'/0'/0'

Compte #1: m/44'/0'/1'

Compte #2: m/44'/0'/2'

Compte #3: m/44'/0'/3'
