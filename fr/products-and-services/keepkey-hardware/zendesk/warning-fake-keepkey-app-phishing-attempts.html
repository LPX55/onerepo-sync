Attention : Tentatives de hameçonnage avec des fausses applications KeepKey

Faites attention aux applications KeepKey frauduleuses dans le Chrome Store. Il est essentiel d'utiliser uniquement le logiciel client officiel pour votre KeepKey : La plateforme ShapeShift. Nous ne vous demanderons jamais votre phrase de graine de 12 mots, si quelqu'un le fait, c'est une arnaque. Les tentatives d'hameçonnage, telles que ces fausses applications de porte-monnaie, peuvent conduire les utilisateurs à perdre leur crypto-monnaie.

La plateforme ShapeShift

La plateforme ShapeShift est l'interface de pointe recommandée pour KeepKey. Visitez beta.shapeshift.com pour connecter votre KeepKey. En fonction de votre KeepKey, vous pouvez être invité à mettre à jour votre bootloader et votre firmware en téléchargeant l'application KeepKey Updater.

Cliquez ici pour une présentation complète de la configuration de votre KeepKey sur la plate-forme ShapeShift.


Le Client KeepKey

Le Client KeepKey est disponible dans le Chrome Web Store. C'était le client original pour interagir avec votre porte-monnaie matériel KeepKey. Cependant, à partir de juin 2020, Google a cessé de prendre en charge les applications Chrome. Nous recommandons à tous les utilisateurs du client KeepKey Chrome App de passer à la plateforme ShapeShift. Cette plateforme vous permet d'envoyer, de recevoir, d'échanger et de conserver en toute sécurité vos actifs KeepKey.

 

Attention, il existe de fausses applications Chrome KeepKey disponibles au téléchargement, alors assurez-vous de n'utiliser que la plateforme ShapeShift pour votre KeepKey.

 

Vous pouvez signaler une fausse application sur le Chrome Store ici. 


Restez en sécurité et n'oubliez pas de télécharger et de connecter votre KeepKey qu'aux applications recommandées ci-dessus. Pour en savoir plus sur la protection contre les tentatives de phishing, cliquez ici.
