如何在 Ubuntu 上获取我的 xPub？

要使用 Ubuntu 在 KeepKey 上获取 xPub，请完成以下步骤。

I. 安装 pip

- 打开终端。
- 输入以下命令 - sudo easy_install pip

II. 安装Cython

- 打开终端。
- 输入以下命令 - sudo pip install cython

III. 安装 Libsub

- 打开终端。
- 输入以下命令 - sudo pip install libusb-1.0.0-dev

IV. 安装 KeepKey Python Client

- 打开终端。
- 输入以下命令 - sudo pip install HTTPS://GITHUB.COM/KEEPKEY/PYTHON-KEEPKEY/ARCHIVE/MASTER.ZIP


V. 检索 xpub 密钥

- 打开终端。
- 输入以下命令 - keepkeyctl get_public_node -n "44'/0'/0'"
- 根据 KeepKey 屏幕和命令提示符中的数字位置输入PIN码
