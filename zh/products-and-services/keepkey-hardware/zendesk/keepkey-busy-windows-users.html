KeepKey 繁忙 - Windows 用户

如果您尝试将 KeepKey 连接到 beta.shapeshift.com 但看到"应用程序冲突"或"KeepKey 繁忙"，您可以尝试以下故障排除步骤：

1. 确保您使用的是 Google Chrome 或 Brave 浏览器，并且您只打开了 beta.shapeshift.com 一个窗口，并且没有其他程序试图连接KeepKey。该设备一次只能连接到一个程序。

如果您已尝试步骤 1，但仍然看到"KeepKey Busy"，并且正在使用 Windows 计算机，请按照以下说明操作：

2. 在 Chrome 或 Brave 浏览器上，点击网址左侧的锁图标。然后单击USB设备旁边的 X。

3. 按Windows键并键入"USB设置"。

4. 点击"蓝牙和其他设备"。

5. 找到您的KeepKey，单击它，然后单击"删除设备"。

6. 刷新 beta.shapeshift.com 或重新启动笔记本电脑/PC，然后尝试重新连接 KeepKey。

 

注意：如果您之前使用不同的浏览器（Chrome，Brave，Firefox，Edge，Safari等）将KeepKey与ShapeShift平台配对，请确保按照相同的过程从这些其他浏览器中删除KeepKey并重新启动笔记本电脑/PC，然后再尝试使用Google Chrome和Brave再次配对KeepKey，这是ShapeShift平台上唯一支持的浏览器。

 

如果您没有看到"KeepKey Busy"消息，但需要获取有关KeepKey设置的帮助，请按照 此处 列出的步骤操作。仍然需要帮助？我们的客户支持团队可以在 此处 找到。
